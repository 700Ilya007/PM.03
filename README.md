Содержательная постановка задачи.

Описать тип (структуру или класс) Fish для хранения данных о продаваемой рыбе (вид рыбы - строка, производитель - строка, цена - строка). Создать класс FishControl, в котором обеспечить хранение одномерного массива указателей (ссылок) на экземпляры типа Fish (специфику языка программирования учесть самостоятельно). В классе FishControl реализовать (в виде отдельных методов) алгоритм сортировки по убыванию (по сочетанию значений двух свойств в порядке убывания приоритета: «вид рыбы» и «цена») и сохранение массива в файл. Формат файла выбрать самостоятельно. В главной функции создать объект класса FishControl и заполнить массив в диалоговом режиме (предварительно запросив размер массива у пользователя). Далее отсортировать массив. Вывести результат в файл.
